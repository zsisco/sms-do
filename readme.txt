To run this program you must have Python 2.7 installed.
Run with >> python smsTK3.py

To set up the userInfo.txt file:
enter in this order...
  email address
  imap server address (e.g. imap.gmail.com)
  folder in inbox that the program looks for emails (use inbox for default)
  phone number (1112223333)

The following commands are valid

1. define <followed by a word>
use this for word definitions

  ex. define computer

2. weather <followed by zip code>
use this for weather forecast

  ex. weather 45701
